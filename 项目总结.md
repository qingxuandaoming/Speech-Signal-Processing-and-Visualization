### ✅ 已完成的功能模块

1. **音频采集模块**
   - 基于PyAudio实现跨平台音频输入
   - 支持16kHz采样率、16位深度
   - 实时音频流处理

2. **信号处理模块**
   - 短时能量计算
   - 过零率计算
   - 短时自相关函数
   - 平均幅度差函数
   - 预加重处理
   - 信号分帧和加窗

3. **语音活动检测**
   - 基于双门限的检测算法

   - 实时语音/静音分类

   - 基于PyQtGraph的动态图表

5. **系统架构**

## 使用说明

### 🚀 快速开始

```bash
# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# 安装依赖
pip install -r real_time_voice_processing/requirements.txt

# 运行系统
python -m real_time_voice_processing.main

# 运行测试
pytest -q

# 运行演示
python -m real_time_voice_processing.demo
```

### 🔄 可复现实验步骤

- 激活虚拟环境并安装依赖：`pip install -r real_time_voice_processing/requirements.txt`
- 运行单元测试以验证实现：`pytest -q`
- 启动演示程序观察能量/ZCR/VAD：`python -m real_time_voice_processing.demo`
- 在 UI 中切换“自适应VAD/自动校准”选项，比较阈值与判决的变化。

### 📱 界面操作

1. **开始处理**：点击"开始处理"按钮启动音频采集和处理
2. **停止处理**：点击"停止处理"按钮停止系统运行
3. **保存数据**：点击"保存数据"按钮保存处理结果到文件
